## Aprendizados
- Frontend TanStack Start com `@cloudflare/vite-plugin` **só roda em Cloudflare Workers** — Vercel serve como estático e dá 404. Deploy = `npx wrangler deploy`, não Vercel.
- Deriv (dashboard novo api.deriv.com) emite **App ID alfanumérico**, não numérico como o formato legado.
- `iqoptionapi` não tem timeout — qualquer chamada síncrona dela no meio de um `lifespan`/request trava o processo inteiro se a rede estiver bloqueada. Sempre rodar em thread com prazo.
- Uma pasta sem `__init__.py` na raiz do projeto com o mesmo nome de um pacote pip (`supabase/`) vira um *namespace package* e engana checks como `try: import x except ImportError`. O check certo é importar o símbolo que você realmente usa.
